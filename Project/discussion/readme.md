### Materials from LQR discussion

 - run the drone simulation with RigidBody.py, which contains 6 dof rigid body dynamics in an openai gym wrapper
 - QuadActuation.py contains lqr implementation and drone control code
 - RigidBodyRender does the animation
 - drone_lqr_math.m contains optimal k matrix derivation, which is used in QuadActuation.py
 - simple_lqr.m contains a simpler example of using lqr
 - state_space.m contains a simple example of state space form and its advantages
 - lqr_notex.docx contains some notes on state space form and lqr theory
